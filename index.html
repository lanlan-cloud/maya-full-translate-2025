Funcionamiento

La aplicación utiliza la API de captura de pantalla para superponerse sobre otras ventanas. Mediante tecnología OCR, detecta texto en la pantalla y lo envía a los servicios de traducción, mostrando los resultados en una interfaz transparente no intrusiva.

Instalación

El proyecto está disponible como extensión de navegador para Chrome y Firefox. Próximamente estará disponible como aplicación desktop para Windows, MacOS y Linux.
